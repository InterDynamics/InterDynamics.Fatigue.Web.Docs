# FAID Quantum Web App

Welcome to the FAID Quantum Web App documentation.

This guide is organised by workflow so users can quickly find the task they want to complete.

## In this guide

- **Getting started** covers account creation, first login, language selection, and groups.
- **Settings** covers theme, language, profile, date and time formatting, and role-based group management.
- **Assessments** covers creating assessments, entering schedules, importing schedules, quick entry, consolidation, and analysis.
- **Analysis outputs** explains dashboards, summaries, schedules, plots, and sleep outputs.
- **Related documents** lists supporting documents referenced by the current guide.

## Quick start

To get started with FAID Quantum Web App:

1. Create an account
2. Create or join a group
3. Create an assessment
4. Enter a work schedule
5. Analyse the work schedule
