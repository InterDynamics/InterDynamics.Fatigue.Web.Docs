# Getting started

To access FAID Quantum Web App, go to `https://faidquantum.com`. The application can be accessed from a smartphone, tablet, or computer.

!!! tip
    On a smartphone, add FAID Quantum Web App to the home screen for quicker access and improved usability.

## Create an account

If you have received an email invitation to join a group, create your account using the same email address as the invitation.

To create an account:

1. Select **Start using FAID Quantum**
2. Select **Sign up** or continue with **Google** or **Microsoft**
3. Enter your email address and password
4. Select **Continue**

## First-time login

### License agreement

The first time you log in, FAID Quantum presents a licence agreement.

1. Read the agreement
2. Select **I Accept the License Agreement**

<!-- IMAGE PLACEHOLDER: License agreement dialog -->
![License agreement placeholder](assets/placeholders/license-agreement.png)

### Language

After accepting the licence agreement, select your language.

!!! note
    The selected language also affects date formatting.

1. Select the appropriate language
2. Select **Continue**

## Groups

Depending on the subscription level, a user may have more than one group attached to their account, or multiple users may access the same group.

Once logged in, the user must create or join a group.

### Accept an invite

If you have been invited to join an existing group, you can either:

1. Enter the **Invite ID** from the email into the welcome screen and select **Submit**
2. Follow the link in the invitation email to accept the invite and join the group

!!! note
    An account must first be created using the specified email address.

<!-- IMAGE PLACEHOLDER: Invite acceptance screen -->
![Accept invite placeholder](assets/placeholders/accept-invite.png)

### Create a group

A group can be created from the welcome screen or from the account menu.

#### From the welcome screen

1. Enter a **Group Name**
2. Select **Next**
3. Select **Create Account**

#### From the account menu

1. Select the top-right account icon
2. Select **Create Group**
3. Enter a **Group Name**
4. Select **Save**
