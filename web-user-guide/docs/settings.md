## Account and display settings

### Theme

Select between:

- **Light**
- **Dark**
- **Auto (system)**

### Language

Select this option to change the language setting.

### Full screen

Select the full-screen option to expand the app view.

- To exit full screen, press **Esc**

### Switch group

If a user has more than one group:

1. Select the account icon
2. Select **Switch Group**
3. Select the desired group

### Profile

Select the account icon, then **Profile**, to view or edit account settings, including:

- Groups
- Invites
- Time and date format
- Logout

<!-- IMAGE PLACEHOLDER: Profile screen -->

#### Groups

The **Groups** section allows the user to view current groups and switch between them.

#### Invites

The **Invites** section shows invites that have been received but not yet accepted. The user can copy the Invite ID and submit it to join the group.

#### Date and time format

1. Select a **Date Format**
2. Select a **Time Format**
3. Select **Save**

## Manage

To manage a group’s settings, select **Manage** from the menu on the left of the page.

### Role

The available group settings depend on the user level:

| Role | Access |
|---|---|
| Group Admin | Can edit settings and users, and delete the group |
| Manager | Can edit settings and users, but cannot delete the group or change its name |
| Group Manager | Can edit and analyse schedules, but cannot modify group defaults |
| User | Does not have access to manage group settings and may only edit their own schedule |

### Manage settings

Select the **Analysis Settings** tab to set fatigue tolerance levels and the default analysis start period.

<!-- IMAGE PLACEHOLDER: Analysis settings screen -->
![Analysis settings placeholder](assets/placeholders/analysis-settings.png)

#### Tolerance level type

1. **None**  
   No fatigue tolerance levels are set, so results will not be coloured by tolerance level.

2. **One**  
   One FAID Score tolerance level and one KSS tolerance level are set.

3. **Multiple**  
   Three fatigue tolerance levels can be set based on task risk. This setting is intended for advanced users.

!!! warning
    The tolerance levels you set are your responsibility.

#### Start period type

The default analysis start date can be set to:

- **X days from first schedule date**
- **X days back from the current date**
- **X days back from the last shift in the schedule**

Then set the number of days for the selected rule.

!!! note
    Seven full days of prior work history are required for accurate fatigue scores. Ensure seven full days of work history are provided before the analysis start date.
