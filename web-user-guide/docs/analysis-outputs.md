# Analysis outputs

The output tabs provide details of the fatigue analysis in relation to the selected tolerance levels.

<!-- IMAGE PLACEHOLDER: Output tabs overview -->
![Outputs overview placeholder](assets/placeholders/outputs-overview.png)

## Navigation

### KSS and FAID Score results

On some outputs, the user can toggle between:

- **FAID Score**
- **KSS**

### Sort by

The user can sort tables by selecting the arrow in the relevant column heading.

### Select an ID

The user can select an ID from the dropdown on:

- **Schedule Output**
- **Fatigue Plot**
- **Sleep Schedule**

## Dashboard

The **Dashboard** tab provides a summary of the analysis, including:

- FAID Score compliance percentage
- KSS compliance percentage
- Peak scores
- Apparent tolerance levels
- Percentage of hours worked in each condition

Compliance and peak score results are coloured:

- **red** when above the configured target or tolerance level
- **green** when below it

## Schedule summary

The **Schedule Summary** tab provides a breakdown for each individual in the schedule.

### Filter by ID

The user can filter by typing ID text into **Filter by ID**.

### Select ID

The user can select a row in the table to navigate to the **Schedule Output** tab for that individual.

## Work schedule

The **Work Schedule** tab provides details of each shift in the analysed schedule.

### Columns displayed

1. Select **Columns** to adjust the information displayed
2. Check or uncheck boxes to show or hide columns
3. Drag columns to reorder them
4. Apply the changes

## Schedule output

The **Schedule Output** tab provides a breakdown of each shift in the schedule, including:

- Peak score
- Time spent in each condition

## Fatigue plot

The **Fatigue Plot** tab provides a plot view of the schedule, coloured according to the peak condition reached in each shift.

For KSS results, predicted sleep is also shown:

- dark grey for night sleep
- light grey for day sleep

Users can:

- hover over a point for more detail
- zoom in on a section of the graph

## Gantt plot

The **Gantt Plot** shows all shifts for all individuals in the schedule, coloured by the peak score of each shift.

### Label display

The user can display labels on each shift, such as peak score.

Zooming in may be required to view the extra detail clearly.

### Show sleep

When viewing KSS results, select **Show Sleep** to display predicted sleep on the graph.

### ID count

Select how many IDs are shown on one screen, then use the arrows to move through the remaining IDs.

### KSS and FS view

When KSS and FAID Score are shown together on the Gantt plot:

- KSS is shown in the top half of the shift block
- FAID Score is shown in the bottom half

### Hover

The user can hover over a shift block to view further details of that shift.
